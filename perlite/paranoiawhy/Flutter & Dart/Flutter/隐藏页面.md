#Flutter 

# `Opacity`

```dart
Opacity(
	opacity: 0,
	child: Foo(),
);
```

# `Offstage`

```dart
Offstage(
	offstage: true,
	child: Foo(),
);
```

# `Visibility`

```dart
Visibility(
	visible: false,
	child: Foo(),
);
```

# `if (condition)`

```dart
if (visible) {
	childOne
} else {
	childTwo
}
```
